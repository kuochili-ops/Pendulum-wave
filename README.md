# 𓃥 白六擺盪球 (Pendulum Wave 3D Simulation)

一個基於 WebGL 與 Three.js 開發的 3D 模擬裝置。本專案精確模擬了經典的**單擺波（Pendulum Wave）**物理現象，並加入了現代網頁互動介面與豐富的視覺主題，讓使用者能從各種黃金視角與多維度的參數調整中，專注欣賞幾何與物理規律交織而成的波動之美。

## 🌟 項目特色

- **完美物理懸吊仿真**：擺索與球體真正具備空間縱深距離，以水平橫桿下方不同的精確物理長度（L）垂直垂吊，呈現完美的流暢階梯斜線。
- **橫向左右擺盪運動**：嚴格對齊真實單擺波裝置，垂球垂直於黑桿身向左、右兩側進行大幅度晃動，在側面視角下可完美觀察到錯落交織的蛇行波動。
- **黃金初始視角**：網頁一打開即自動定位在最理想的「側面仰視角」，並預設自動收合操作面板，呈現全螢幕專注欣賞的洗鍊畫面。
- **智慧收合操作介面**：
  - 支援點擊面板右上角按鈕進行收合與展開。
  - **直覺互動**：在介面開啟狀態下，只要點擊或觸碰 3D 擺盪顯示區（Canvas），介面便會自動智慧收合，流暢不擋視線。
- **手指/滑鼠滑動操控**：全面支援行動裝置單指滑動（電腦按住左鍵拖曳）旋轉任意觀看角度，以及雙指縮放視角。

## ⚙️ 互動控制功能

操作面板提供以下即時調校參數：

1. **選擇擺盪模式**：
   - `標準單擺波`：同向同側釋放，隨時間自動演變成規律的蛇行與交錯波浪。
   - `交錯鋸齒`：正反兩側交錯釋放，開場即呈現極具視覺衝擊的網格律動。
   - `中央擴散`：從中心向兩側加入空間相位漸變，如水滴般向外擴散。
   - `混沌樂章`：隨機初始化相位，呈現完全無序的無規律運動。
2. **球體色彩主題**：內建多款高對比與漸層視覺主題：
   - 經典彩虹漸層、純奢華金 (Metallic Gold)、純太空銀 (Metallic Silver)、翡翠綠漸層、極光藍紫漸層、賽博霓虹漸層。
3. **擺盪角度 (幅度) 拉桿**：可即時調整擺動幅度（10° 至 60°）。
4. **擺索增長比例拉桿**：等比例一律加長或縮短所有垂球的索長，不破壞原本完美的斜線排列。
5. **擺盪速度快慢拉桿**：打破長繩擺動必變慢的物理限制，內建時間流速增強器（1.0x 至 5.0x），免去漫長等待，即時看完完整演變週期。
6. **動態控制按鈕**：提供「開始/停止」凍結功能與「重新釋放」歸零鈕。

## 🔬 物理原理簡介

本裝置背後的核心物理基於單擺的週期公式：

$$T = 2\pi \sqrt{\frac{L}{g}} \implies L = g \left(\frac{T}{2\pi}\right)^2$$

在一個固定的循環週期 $T_{loop}$（本專案設定為 60 秒）內，我們讓第一根長繩到最後一根短繩的擺盪次數依序遞增（如每分鐘各自完成 14、15、16... 次擺動）。由於每根擺繩被分配到的週期 $T$ 不同，經由重力加速度 $g$ 計算出的擺長 $L$ 便會形成一條優美的階梯斜線。當它們同時釋放時，角頻率的微小差異便會在空中組合出炫目的波浪相差圖案。

## 🛠️ 技術架構

- **開發語言**：HTML5 / JavaScript (ES6+)
- **3D 引擎**：[Three.js (r128)](https://threejs.org/)
- **視角控制**：Three.js OrbitControls
- **渲染技術**：WebGL 渲染器、PCFSoftShadowMap 柔和陰影計算、FogExp2 指數霧氣環境營造。

## 🚀 快速開始

本項目為純前端網頁應用，無需安裝任何後端依賴環境：

1. 下載專案中的 `index.html`。
2. 在任意現代瀏覽器（如 Chrome, Safari, Edge）中直接雙擊打開該檔案。
3. 使用手機觸控或電腦滑鼠即可開始體驗物理與視覺之美。
