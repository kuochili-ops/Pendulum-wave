# 𓃥 白六擺盪球 (Pendulum Wave 3D Audio-Visual Simulation)

一個結合 WebGL 3D 視覺與 Web Audio API 的精緻動態裝置。本專案不僅精確模擬了經典的**單擺波（Pendulum Wave）**物理現象，更將其轉化為一個空靈的「視覺樂器」，讓垂球在空中規律交錯的同時，編織出迷人的水晶音樂盒樂章。

## 🌟 項目特色

- **聲學幾何聯動（繩越短音越高）**：當垂球擺盪至左側或右側最高處（物理速度為 0、振幅最大的瞬間）時，會精確觸發清脆的音符。音階對齊經典的**五聲音階（Pentatonic Scale）**，長繩負責沉穩的低音，短繩則負責清脆的高音。
- **空間立體聲聲道（Stereo Panning）**：即時計算每個垂球在 3D 場景中的 Z 軸空間位置。當垂球偏向左側最高點時，聲音會偏向左耳；偏向右側時則偏向右耳，配戴耳機觀賞時能獲得極佳的環繞立體聲體驗。
- **完美物理懸吊仿真**：擺索與球體真正具備空間縱深距離，以水平橫桿下方不同的精確物理長度（L）垂直垂吊，靜止時呈現流暢的前長後短階梯斜線。
- **橫向左右擺盪運動**：嚴格對齊真實單擺波裝置，垂球垂直於黑桿身向左、右兩側進行大幅度晃動，在側面視角下可完美觀察到錯落交錯的蛇行波動。
- **黃金初始視角**：網頁一打開即自動定位在最理想的「側面仰視角」，並預設自動收合操作面板，呈現全螢幕專注欣賞的洗鍊畫面。
- **智慧收合操作介面**：
  - 支援點擊面板右上角按鈕進行手動收合與展開。
  - **直覺互動**：在介面開啟狀態下，只要點擊或觸碰 3D 擺盪顯示區（Canvas），介面便會自動智慧收合，流暢不擋視線。

## ⚙️ 互動控制功能

操作面板提供以下即時調校參數：

1. **選擇擺盪模式**：
   - `標準單擺波`：同向同側釋放，隨時間自動演變成規律的蛇行與交錯波浪。
   - `交錯鋸齒`：正反兩側交錯釋放，開場即呈現極具視覺衝擊的網格律動。
   - `中央擴散`：從中心向兩側加入空間相位漸變，如水滴般向外擴散。
   - `混沌樂章`：隨機初始化相位，呈現完全無序的無規律運動。
2. **球體色彩主題**：內建多款高對比與漸層視覺主題：
   - 經典彩虹漸層、純奢華金 (Metallic Gold)、純太空銀 (Metallic Silver)、翡翠綠漸層、極光藍紫漸層、賽博霓虹漸層。
3. **擺盪角度 (幅度) 拉桿**：可即時調整擺動幅度（10° 至 60°）。
4. **擺索增長比例拉桿**：等比例一律加長或縮短所有垂球的索長，不破壞原本完美的斜線排列。
5. **擺盪速度快慢拉桿**：內建時間流速增強器（1.0x 至 5.0x），加快時間流速時，音效的節奏也會同步成倍加快，免去漫長等待，即時看完完整演變週期。
6. **動態控制按鈕**：提供「開始/停止」凍結功能與「重新釋放」歸零鈕。

## 🔬 物理與聲學原理

本裝置背後的核心物理基於單擺的週期公式：

$$T = 2\pi \sqrt{\frac{L}{g}} \implies L = g \left(\frac{T}{2\pi}\right)^2$$

在一個固定的循環週期 $T_{loop}$（本專案設定為 60 秒）內，我們讓第一根長繩到最後一根短繩的擺盪次數依序遞增（如每分鐘各自完成 14、15、16... 次擺動）。由於每根擺繩被分配到的週期 $T$ 不同，經由重力加速度 $g$ 計算出的擺長 $L$ 便會形成一條優美的階梯斜線。當它們同時釋放時，角頻率的微小差異便會在空中組合出炫目的波浪相差圖案。

在聲學表現上，為了防止 16 個音符同時發聲時產生刺耳的雜音，音訊包絡線設計了極快的擊弦速度（Attack）與 0.6 秒的指數型音量衰減（Decay Envelope），讓聲音如同純淨的水晶音樂盒或鋼琴一樣具有清脆的顆粒感。

## 🛠️ 技術架構

- **開發語言**：HTML5 / JavaScript (ES6+)
- **3D 引擎**：[Three.js (r128)](https://threejs.org/)
- **音訊引擎**：瀏覽器原生 Web Audio API (無任何外部音檔依賴，純代碼即時合成波形)
- **視角控制**：Three.js OrbitControls
- **渲染技術**：WebGL 渲染器、PCFSoftShadowMap 柔和陰影計算、FogExp2 指數霧氣環境營造。

## 🚀 快速開始

本項目為純前端網頁應用，無需安裝任何後端依賴環境：

1. 下載專案中的 `index.html`。
2. 在任意現代瀏覽器（如 Chrome, Safari, Edge）中直接雙擊打開該檔案。
3. 由於瀏覽器的隱私安全規範限制，音訊無法自動播放，請點擊畫面中央的**「開啟 𓃥 白六擺盪球 (含音效)」**按鈕。
4. 使用手機觸控或電腦滑鼠滑動即可開始調整視角與參數，體驗物理、視覺與聽覺交織的洗滌心靈之美。
